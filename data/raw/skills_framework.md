# Skills Framework — Career Intelligence Knowledge Base

## Skill Categories

### 1. Technical Skills

#### Programming Languages
- **Python**: General-purpose language widely used in data science, ML, web development, and automation. Proficiency levels: basic scripting, intermediate (OOP, testing), advanced (system design, performance).
- **JavaScript/TypeScript**: Primary language for web development (frontend and backend via Node.js). Essential for full-stack roles.
- **SQL**: Structured Query Language for database querying and management. Required for virtually all data roles.
- **R**: Statistical programming language popular in academia and biostatistics. Alternative to Python for data analysis.
- **Java**: Enterprise-grade language for backend systems, Android development, and large-scale applications.

#### Data and Analytics
- **Data Visualization**: Creating charts, dashboards, and visual representations of data. Tools: Tableau, Power BI, matplotlib, D3.js.
- **Statistical Analysis**: Hypothesis testing, regression, probability distributions, experimental design.
- **Machine Learning**: Supervised learning, unsupervised learning, deep learning, model evaluation and selection.
- **Data Engineering**: ETL pipelines, data warehousing, stream processing, data quality.
- **Big Data Technologies**: Spark, Hadoop, distributed computing concepts.

#### Cloud and Infrastructure
- **AWS**: Amazon Web Services — EC2, S3, Lambda, SageMaker, RDS.
- **GCP**: Google Cloud Platform — BigQuery, Vertex AI, Cloud Run, GKE.
- **Azure**: Microsoft Azure — Azure ML, Cosmos DB, AKS, Azure Functions.
- **Docker**: Containerization for reproducible deployments.
- **Kubernetes**: Container orchestration for scalable systems.

#### Cybersecurity
- **Network Security**: Firewalls, VPNs, intrusion detection systems.
- **Application Security**: OWASP Top 10, secure coding practices, vulnerability assessment.
- **Identity and Access Management**: Authentication, authorization, SSO, MFA.
- **Incident Response**: Threat detection, forensics, recovery procedures.

### 2. Cognitive Skills

- **Analytical Thinking**: Breaking complex problems into components, identifying patterns, evaluating evidence.
- **Creative Thinking**: Generating novel solutions, brainstorming, design thinking approaches.
- **Critical Thinking**: Evaluating arguments, identifying biases, making evidence-based decisions.
- **Systems Thinking**: Understanding how components interact within larger systems.
- **Problem Solving**: Structured approaches to identifying and resolving challenges.

### 3. Interpersonal Skills

- **Communication**: Written and verbal communication, presentation skills, active listening.
- **Collaboration**: Cross-functional teamwork, conflict resolution, consensus building.
- **Leadership**: Mentoring, delegation, vision setting, motivating teams.
- **Stakeholder Management**: Managing expectations, influence without authority, negotiation.
- **Emotional Intelligence**: Self-awareness, empathy, social skills, relationship management.

### 4. Self-Management Skills

- **Resilience**: Adapting to change, recovering from setbacks, maintaining performance under pressure.
- **Curiosity and Lifelong Learning**: Proactive skill development, staying current with industry trends.
- **Time Management**: Prioritization, focus, meeting deadlines effectively.
- **Self-Motivation**: Internal drive, goal setting, accountability.

## Skill Proficiency Levels

| Level | Description | Indicators |
|-------|-------------|------------|
| Beginner | Basic awareness and foundational knowledge | Can follow tutorials, understands concepts |
| Intermediate | Can apply skills independently | Completes tasks without guidance, handles common scenarios |
| Advanced | Deep expertise, can teach others | Handles complex scenarios, optimizes solutions, mentors |
| Expert | Industry-recognized authority | Innovates, sets standards, leads strategic decisions |

## Role-to-Skill Mapping

### Data Analyst
- **Must have**: SQL (intermediate), Python or R (intermediate), Data Visualization (intermediate), Statistical Analysis (intermediate), Communication (intermediate)
- **Nice to have**: Machine Learning (beginner), Cloud platforms (beginner), Data Engineering (beginner)

### ML Engineer
- **Must have**: Python (advanced), Machine Learning (advanced), Software Engineering (advanced), MLOps (intermediate), Mathematics (intermediate)
- **Nice to have**: Cloud platforms (intermediate), Data Engineering (intermediate), System Design (intermediate)

### Product Manager
- **Must have**: Communication (advanced), Analytical Thinking (advanced), Strategic Thinking (advanced), Stakeholder Management (intermediate), Data Analysis (intermediate)
- **Nice to have**: Technical Literacy (intermediate), User Research (intermediate), SQL (beginner)

### Information Security Analyst
- **Must have**: Network Security (intermediate), Application Security (intermediate), Incident Response (intermediate), IAM (intermediate)
- **Nice to have**: Cloud Security (intermediate), Programming (intermediate), Risk Management (intermediate)
