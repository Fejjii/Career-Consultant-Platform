## Skills Disruption

By 2030, approximately 39% of existing core skills will be transformed or become obsolete. Workers will need to reskill or upskill to remain competitive.

The most in-demand skill categories are:
- Technology skills (particularly AI/ML and data analysis)
- Cognitive skills (analytical and creative thinking)
- Self-efficacy skills (resilience, curiosity, lifelong learning)

## Data Analyst Role Profile

**Alternative titles:** Data Analyst, Business Analyst (Data), Junior Data Scientist

**Core responsibilities:**
- Collect, clean and analyze datasets
- Create visualizations and dashboards
- Generate insights from data to support business decisions
- Write SQL queries and use statistical tools
- Communicate findings to non-technical stakeholders

**Required skills:**
- SQL and database management
- Python or R for data analysis
- Data visualization (Tableau, Power BI, or similar)
- Statistical analysis and hypothesis testing
- Communication and storytelling with data
- Excel/spreadsheet proficiency
- Basic machine learning concepts (nice to have)

**Typical career progression:**
- Junior Data Analyst → Data Analyst → Senior Data Analyst → Lead Analyst / Analytics Manager
- Alternative paths: Data Scientist, Business Intelligence Engineer, Product Analyst

## Machine Learning Engineer Role Profile

**Alternative titles:** ML Engineer, Applied ML Scientist, AI Engineer

**Core responsibilities:**
- Design and implement ML models for production systems
- Build and maintain ML pipelines (training, evaluation, deployment)
- Optimize model performance and scalability
- Collaborate with data scientists and software engineers
- Monitor model performance in production

**Required skills:**
- Python (advanced)
- Machine learning frameworks (PyTorch, TensorFlow, scikit-learn)
- Software engineering best practices
- MLOps and deployment (Docker, Kubernetes, CI/CD)
- Mathematics (linear algebra, probability, optimization)
- Cloud platforms (AWS, GCP, or Azure)
- Data engineering basics (SQL, ETL pipelines)

**Typical career progression:**
- Junior ML Engineer → ML Engineer → Senior ML Engineer → Staff ML Engineer / ML Architect
- Alternative paths: Research Scientist, MLOps Engineer, AI Product Manager

## Product Manager Role Profile

**Core responsibilities:**
- Define product vision and strategy
- Prioritize features based on user needs and business goals
- Work with engineering, design, and stakeholders
- Analyze metrics and user feedback
- Manage product roadmap and releases

**Required skills:**
- Strategic thinking and prioritization
- User research and empathy
- Data analysis and metrics literacy
- Communication and stakeholder management
- Technical literacy (understanding APIs, databases, architecture)
- Agile methodologies
- Market analysis

**Typical career progression:**
- Associate PM → Product Manager → Senior PM → Director of Product → VP Product / CPO

## Program Manager Role Profile

**Core responsibilities:**
- Coordinate multiple related projects toward a common goal
- Manage cross-functional dependencies and timelines
- Risk management and mitigation
- Stakeholder communication and reporting
- Process improvement and governance

**Required skills:**
- Project management methodologies (Agile, Waterfall)
- Risk management
- Stakeholder communication
- Budget management
- Resource allocation
- Technical understanding of program domain
- Leadership and conflict resolution

**Typical career progression:**
- Project Manager → Program Manager → Senior Program Manager → Director of Program Management
